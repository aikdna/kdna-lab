# Prompt Engineering Diagnosis

## 1. 核心意图分析

**显式意图**：用户要求AI同时完成四项独立任务——撰写气候变化报告、分析Excel数据、撰写客户邮件、规划下周行程。

**隐含假设**：用户假设AI可以同时处理这些任务，且输出结果会自然整合。实际上，这些任务之间**没有逻辑关联**，属于典型的“多任务混杂”问题。

## 2. 任务模糊性检测

| 问题 | 严重程度 |
|------|----------|
| **多目标混合**：四项任务彼此独立，无优先级或关联说明 | ⚠️ **功能性** |
| **成功标准缺失**：报告长度/风格？Excel数据类型？邮件收件人？行程约束？ | ⚠️ **功能性** |
| **执行顺序未定义**：AI可能随机选择一项开始，或尝试一次性输出所有内容 | ⚠️ **功能性** |

## 3. 缺失上下文清单

- **气候变化报告**：目标受众？报告用途（学术/政策/科普）？字数要求？是否需要数据引用？
- **Excel表格**：表格未提供！AI无法分析不存在的文件。
- **客户邮件**：客户身份？邮件目的（跟进/投诉/提案）？语气要求（正式/友好）？
- **下周行程**：当前日期？已有安排？优先级（工作/个人）？时间粒度（小时/天）？

## 4. 输出格式分析

**问题**：未指定任何格式要求。AI可能：
- 输出一个混乱的段落混合所有内容
- 随机选择一项任务完成
- 要求用户澄清（但用户期望直接执行）

**风险**：用户可能认为AI“不听话”或“能力不足”，实际是提示词结构缺陷。

## 5. 风险措辞检测

| 风险类型 | 具体表现 | 严重程度 |
|----------|----------|----------|
| **过度宽泛** | “写一篇报告” → 无范围限制，AI可能写10万字或100字 | ⚠️ **功能性** |
| **隐含矛盾** | 同时要求“分析数据”和“写报告”，但未说明数据是否与报告相关 | ⚠️ **功能性** |
| **执行不可能** | “分析Excel表格” → 表格未提供，AI无法执行 | 🔴 **安全-功能失效** |

## 6. 具体改写方案

### 原始提示（有缺陷）
> 请帮我分析这个prompt为什么产生的输出不理想："写一篇关于气候变化的报告，同时帮我分析这个Excel表格的数据，然后写一封邮件给客户，最后帮我规划下周的行程。"

### 改写版本（修复后）

```
请按以下顺序依次执行四个独立任务。每个任务完成后，请明确标注任务编号和标题，再开始下一个任务。

【任务1：撰写气候变化报告】
- 目标受众：公司管理层（非技术背景）
- 字数：800-1000字
- 重点：2023-2024年全球气候政策变化及对企业的影响
- 格式：分三段（现状、趋势、建议）

【任务2：分析Excel数据】
- 数据文件：请先上传Excel文件（.xlsx或.csv格式）
- 分析要求：描述数据集的列名、行数、关键统计量（均值/中位数/异常值）
- 输出：一个简洁的数据摘要表格

【任务3：撰写客户邮件】
- 收件人：张总（长期合作客户）
- 目的：通知产品价格调整（上涨5%），并安抚客户
- 语气：专业且友好
- 要求：包含感谢语、调价原因说明、后续沟通建议

【任务4：规划下周行程】
- 当前日期：2025年3月17日（周一）
- 已有固定安排：周二上午10点团队会议，周四下午2点客户拜访
- 优先级：工作事项优先于个人事项
- 输出格式：按天列出，每天分为“上午/下午/晚上”三个时段

请确认收到后开始执行。如果缺少信息（如Excel文件），请先询问再继续。
```

### 每个修复解决的问题

| 修改点 | 解决的问题 | 对应缺陷编号 |
|--------|------------|--------------|
| 明确执行顺序 | 消除多任务混杂的歧义 | 2 |
| 为每项任务添加具体参数 | 提供缺失的上下文 | 3 |
| 指定输出格式（分任务标注） | 避免输出混乱 | 4 |
| 要求先上传文件再分析 | 解决“无法分析不存在的文件”问题 | 5 |
| 添加确认机制 | 防止AI跳过关键步骤 | 5 |

## 7. 严重性评级总结

| 缺陷 | 严重程度 | 说明 |
|------|----------|------|
| 多任务混杂 | ⚠️ 功能性 | 导致AI无法确定执行策略 |
| 缺失Excel文件 | 🔴 安全-功能失效 | 任务2完全无法执行 |
| 无输出格式要求 | ⚠️ 功能性 | 输出可能不符合用户预期 |
| 无上下文参数 | ⚠️ 功能性 | 报告/邮件/行程质量不可控 |
| 无优先级/顺序 | ⚠️ 功能性 | AI可能随机选择任务 |

**最终诊断**：该提示词的主要问题是**结构混乱**和**上下文缺失**，而非AI能力不足。通过将单一提示拆分为四个独立子任务，并为每个任务提供明确参数，可显著提升输出质量。
