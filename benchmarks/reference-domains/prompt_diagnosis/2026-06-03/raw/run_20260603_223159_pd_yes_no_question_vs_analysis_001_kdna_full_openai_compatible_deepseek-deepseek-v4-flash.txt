这个 prompt 的核心问题在于 **目标模糊** 和 **上下文缺失**，而不是格式或任务混合。

**具体诊断：**

1. **目标模糊**：你只问了一个宽泛的问题“AI会取代人类工作吗？”，但没有说明你希望得到什么类型的回答。是想要一个客观的趋势分析？一个支持或反对的论点？一个针对特定行业（如制造业、创意行业）的预测？还是一个面向普通读者的科普解释？不同的目标会导向完全不同的输出结构、深度和立场。目前这个 prompt 没有定义“完美输出”的样子——你无法用一句话描述这个回答能让你下一步做什么。

2. **上下文缺失**：这个问题本身缺乏任何背景信息。比如：
   - 你关注的是短期（未来5年）还是长期（50年）？
   - 你指的是全部工作还是特定岗位？
   - 你是否有特定的地域、经济或技术假设？
   - 你希望回答基于学术研究、行业报告还是个人观点？
   没有这些上下文，模型只能给出最泛泛的通用回答，很可能缺乏针对性和可操作性。

3. **没有任务混合**：这是一个单一问题，没有同时要求分析、生成、格式化等冲突任务，所以不需要拆分。

**建议方向（不要直接加格式或角色）：**

- 先明确你希望这个回答帮你做什么。例如：“我想写一篇面向大学生的科普文章，解释AI对就业的影响，需要平衡乐观和谨慎的观点。” 或者 “我需要为公司的战略会议准备一份10页报告，重点分析AI对客服岗位的替代风险和时间线。”
- 补充必要的上下文：比如行业、时间范围、目标读者、回答的用途（决策参考、学习、辩论等）。

**一句话总结：** 当前 prompt 不是“不好”，而是“不完整”——它缺少让模型知道该往哪个方向深入的核心信息。先定义清楚“完美输出能让你做什么”，再考虑是否要调整格式或添加约束。
