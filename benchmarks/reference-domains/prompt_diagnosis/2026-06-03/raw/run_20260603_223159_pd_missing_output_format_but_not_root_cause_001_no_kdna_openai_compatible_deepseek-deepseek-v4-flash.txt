## Prompt 诊断：`"帮我分析竞争对手"`

### 1. 总体评价：质量较低，过于模糊

该 prompt 缺少关键的上下文信息和具体指令，属于典型的“零意图”提问。AI 无法准确理解你的真实需求，容易给出泛泛而谈的回答。

### 2. 主要问题

| 问题维度 | 具体表现 |
|----------|----------|
| **范围不明** | 没有指定行业、市场、地域或具体竞品名称（如“分析可口可乐”vs“分析瑞幸咖啡”）。 |
| **分析维度缺失** | 未说明需要分析哪些方面（产品功能、定价、营销、用户口碑、财务状况、市场份额等）。 |
| **目标不清晰** | 你是想了解竞争格局？制定策略？写报告？还是做产品优化？不同目标需要的分析深度不同。 |
| **输出格式未指定** | 未说明希望以表格、段落、列表、对比矩阵还是报告形式呈现。 |
| **数据来源假设** | 是否允许 AI 基于公开信息（如财报、行业报告）进行推测？还是仅限你提供的资料？ |

### 3. 改进建议

一个有效的 prompt 应包含以下要素（括号内为示例）：

- **行业/场景**：例如，“我是一家 SaaS 公司的产品经理，所在赛道是**项目管理工具**”。
- **具体竞品**：例如，“主要对比**Notion、Asana、ClickUp**三款产品”。
- **分析维度**：例如，“从**功能特性、定价策略、目标用户、近半年更新亮点**四个方面分析”。
- **输出要求**：例如，“请用**Markdown 表格**呈现对比结果，每个维度下给出2-3句话的简要总结”。

**改进版示例**：
> 我是一家新创项目管理工具的创始人，请帮我分析竞争对手 Notion、Asana 和 ClickUp。从**核心功能、定价模式、目标用户群体、2024年关键更新**四个维度进行对比，输出为 Markdown 表格，并附上一段文字总结各自的优劣势。

## 关于输出格式：是否应该用 Markdown 表格？

### 视场景而定，但非常推荐

| 情况 | 建议格式 | 原因 |
|------|----------|------|
| **对比多个竞品的多项属性** | ✅ **Markdown 表格** | 结构清晰，一目了然，便于横向比较 |
| **单独分析一个竞品的深度报告** | ❌ 段落/列表更合适 | 表格会割裂逻辑，不如分段叙述流畅 |
| **仅需列举竞品名称和一句话定位** | ✅ 表格或无序列表均可 | 表格可额外附带排序、评分等列 |
| **输出包含复杂长文本（如战略分析）** | ❌ 正文+小标题 | 表格不适合容纳大段文字，易导致换行混乱 |

### 你的 prompt 中需要主动要求表格

原始 prompt 没有指定格式，AI 默认会以段落或列表输出。**如果你想要表格，必须明确说明**，例如：

> “请用 Markdown 表格格式输出分析结果。”
