这个 prompt 的核心问题不是格式，而是**目标模糊**和**上下文缺失**。

你只说了“帮我分析竞争对手”，但缺少几个关键信息：
- 分析哪些竞争对手？（行业、具体公司、产品线？）
- 分析什么维度？（市场份额、定价策略、技术优势、营销渠道、用户评价？）
- 分析的目的是什么？（用于制定竞争策略、写报告、还是投资决策？）
- 有没有任何背景数据或限制条件？（比如时间范围、地域、数据来源？）

**输出格式（如 Markdown 表格）只有在明确了分析内容和目标之后才有意义。** 如果现在强行指定表格格式，模型可能会生成一个结构漂亮但内容空洞或偏离你真实需求的表格——这是最危险的失败模式：格式正确，但实质错误。

建议你先用一句话回答这个问题：**“一个完美的分析结果，能让我接下来做什么？”**  
例如：“能让我知道对手在哪些功能上领先，从而调整我们的产品路线图。” 然后基于这个目标，补充具体竞争对手名称、分析维度、以及你已有的数据或假设。

一旦目标清晰、上下文充足，再决定用表格、列表还是段落来呈现，会事半功倍。
