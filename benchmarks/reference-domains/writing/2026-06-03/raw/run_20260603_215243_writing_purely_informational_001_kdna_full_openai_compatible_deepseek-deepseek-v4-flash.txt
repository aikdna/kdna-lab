### 详细接口文档（示例）

#### **`POST /users`**  <a id="post-users"></a>

**说明（Description）**

创建一个新的用户账户。账户创建后，会立即发送一封包含验证链接的邮件。

**请求参数（Request Parameters）**

| 参数名 | 位置 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| `name` | Body | string | 是 | 用户姓名，长度限制为 2-50 个字符。 |
| `email` | Body | string | 是 | 用户邮箱，需为合法格式且未被注册。 |
| `password` | Body | string | 是 | 用户密码，必须包含大写字母、小写字母和数字，至少8位。 |

**请求示例（Request Example）**

```json
{
  "name": "张三",
  "email": "zhangsan@example.com",
  "password": "MyP@ssw0rd"
}
```

**成功响应（Success Response）**

- **状态码**: `201 Created`

| 参数名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `id` | string | 新建用户的唯一标识符（UUID）。 |
| `name` | string | 用户姓名。 |
| `email` | string | 用户邮箱。 |
| `created_at` | string | 账户创建时间（ISO 8601）。 |

**响应示例（Response Example）**

```json
{
  "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "name": "张三",
  "email": "zhangsan@example.com",
  "created_at": "2024-05-20T10:30:00Z"
}
```

**错误响应（Error Response）**

- **状态码**: `400 Bad Request`
    - 当邮箱已被注册时，返回：`{ "error_code": "EMAIL_EXISTS", "message": "The email address is already registered." }`
- **状态码**: `422 Unprocessable Entity`
    - 当参数不符合校验规则时，返回：`{ "error_code": "VALIDATION_ERROR", "message": "Password must contain uppercase and lowercase letters." }`
